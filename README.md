# FlowMind – Frontend
Developed a responsive and interactive user interface using React.js, implementing features such as dark mode, real-time search, and dynamic task filtering
Built reusable components and managed state using React Hooks for efficient UI updates and seamless user experience
Integrated frontend with backend APIs using Axios, enabling real-time task management and authentication workflows
